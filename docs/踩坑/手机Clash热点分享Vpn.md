# 手机Clash热点分享Vpn

### 话不多说开始上车

请确保在已经通过clash正确导入订阅
google.com正常访问后继续

### 配置信息

clash版本: [2.5.12.premium](https://github.com/Kr328/ClashForAndroid/releases/tag/v2.5.12)
运行环境: 手机鸿蒙 3.0.0.165

### clash开启局域网共享

clash设置 > 覆写 > 允许来自局域网的连接


#### 查看本机IP

手机设置 > 关于手机 > 状态信息


### 其他手机如何连接

其他手机连接clash这台手机的热点
在连接热点时在高级选项里面找到手动代理，输入上面的clash主机的IP，端口7890
